## 📚 메인 기능

- 사용자들이 온라인에서 롤링 페이퍼를 만들고, 공유할 수 있도록 다양한 기능을 제공합니다.

- 주요 기능으로는 롤링 페이퍼 생성 및 수정하고 메시지를 보내고 리액션을 남길 수 있습니다.

- 반응형 웹 디자인을 지원하며, 사용자가 롤링 페이퍼에 직접 메시지를 추가하고 친구나 지인들과의 공유 기능이 있습니다.
